【项目简介】
一个玻璃态（Glassmorphism）风格的 Web 互动叙事游戏（Galgame），由 DeepSeek API 驱动，
支持多角色对话、好感度系统、多结局、CG 图鉴、故事导出、TTS 语音自动播放等功能。
用户注册登录后，所有游戏数据（对话历史、好感度、角色配置等）持久化保存到数据库。

【核心功能】
✅ 玻璃态 macOS 风格 UI，平滑动画与毛玻璃效果
✅ 用户系统：注册、登录、登出，数据独立存储
✅ AI 动态生成剧情、角色回复、分支选项
✅ 好感度系统：根据对话内容自动增减，影响结局
✅ 多结局：幸福结局（好感度≥95）、离别结局（好感度≤10）
✅ 多角色管理：可动态添加/切换角色，自定义性格描述
✅ 导出故事：一键将完整对话导出为 TXT 文件
✅ CG 图鉴：解锁并收藏游戏中的特殊场景图片（可扩展）
✅ 自动播放模式：每 3 秒自动发送当前输入框内容
✅ TTS 语音合成：每条角色回复可自动或手动播放语音（需 TTS API）
✅ 响应式设计：适配 PC 与移动设备

【环境要求】
- Web 服务器：Apache / Nginx / IIS
- PHP 版本：7.4 或更高（需开启 curl、pdo_mysql、session）
- MySQL 版本：5.7 或更高
- 浏览器：现代浏览器（Chrome, Edge, Safari, Firefox）

【部署步骤】

1. 准备目录
   将所有文件上传至网站目录，例如：/var/www/html/ai_galgame/

2. 创建数据库
   登录 MySQL，执行 database.sql 中的 SQL 语句：
   mysql -u root -p < database.sql
   或通过 phpMyAdmin 导入。

3. 修改配置文件
   编辑 config.php，填写数据库连接信息与 DeepSeek API 密钥：
   define('DB_PASS', '你的数据库密码');
   define('DEEPSEEK_API_KEY', 'sk-你的DeepSeek密钥');

4. 设置目录权限（可选）
   确保 PHP 会话目录可写，通常无需额外设置。

5. 配置 Web 服务器
   确保网站根目录指向 ai_galgame 文件夹，并开启 PHP 解析。

6. 访问游戏
   在浏览器中打开 http://你的域名/ai_galgame/index.html
   注册账号 → 登录 → 开始游戏。

【API 依赖】
- 对话生成：DeepSeek API（https://platform.deepseek.com/）
  需注册并获取 API Key，填入 config.php。
- 语音合成（TTS）：第三方接口 https://api.milorapart.top/apis/AIvoice/
  无需密钥，若接口失效可自行替换为其他 TTS API（修改 game.php 中的 tts 部分）。

【使用说明】

1. 注册/登录
   - 首次使用需注册用户名和密码（用户名≥3字符，密码≥4字符）。
   - 登录后自动加载之前的游戏进度。

2. 游戏界面
   - 左侧：角色列表，点击切换当前对话角色；可添加新角色。
   - 中间：聊天区域，显示对话气泡、场景提示、AI 生成的选项按钮。
   - 右侧：角色立绘（emoji 占位）、好感度显示、自动播放/自动语音开关。

3. 交互方式
   - 在底部输入框输入文字，按回车或点击“发送”。
   - 点击 AI 生成的选项按钮，自动填入并发送。
   - 点击每条角色消息旁的 🔊 按钮播放该句语音（需网络）。
   - 开启“自动播放”后，每 3 秒自动发送输入框内容（需预先输入文字）。
   - 开启“自动语音”后，每条 AI 回复自动朗读。

4. 存档与导出
   - 游戏状态（对话历史、好感度、角色等）自动保存至数据库。
   - 点击“导出故事”可下载完整对话记录为 TXT 文件。

5. 添加角色
   - 点击左侧“➕ 添加角色”，输入名称、性格描述、初始好感度。
   - 新角色将出现在角色列表中，可随时切换对话。

6. CG 图鉴
   - 通过 game.php 中的 unlock_cg 接口可解锁 CG（需二次开发）。
   - 点击“CG 图鉴”查看已解锁的图片列表。

【常见问题】

Q: 登录后白屏或无法加载？
A: 检查 config.php 数据库配置是否正确，确认 MySQL 服务已启动。
   查看 PHP 错误日志（如 /var/log/php_errors.log）。

Q: AI 回复失败，提示“API接口调用失败”？
A: 检查 config.php 中的 DEEPSEEK_API_KEY 是否正确，网络是否能访问 api.deepseek.com。
   可临时用浏览器访问 https://api.deepseek.com 测试连通性。

Q: 语音无法播放？
A: 确认浏览器允许自动播放音频，且 TTS 服务可用。
   可打开浏览器控制台（F12）查看网络请求是否返回音频 URL。

Q: 如何修改游戏背景、立绘或添加 CG 图片？
A: 背景图通过 CSS 的 body 渐变背景修改；立绘上传功能需扩展后端存储；
   CG 功能已预留数据库表与 API，可在剧情节点调用 game.php?action=unlock_cg 解锁。

【扩展建议】
- 使用 HTTPS 部署以保证语音 API 正常调用（部分浏览器限制混合内容）。
- 将 emoji 立绘替换为真实图片：修改 script.js 中的 char.emoji 为图片 URL。
- 增加更多结局条件：修改 game.php 中 chat 分支的好感度阈值。
- 集成 Stable Diffusion 或 DALL·E 生成 CG：替换 unlock_cg 中的图片生成逻辑。
